HTML tables should be used to display data relating to objects that have the same properties. In other words, HTML tables serve a functional purpose (organizing data) rather than for layouts.

Basic Table Tags in HTML

  • <table> - Specifies a table
  • <caption> - Specifies a table caption
  • <tr> - Specifies a table row
  • <th> - Specifies a table header
  • <td> - Specifies a table cell / detail
